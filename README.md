# Data Structures & Algorithms (DS_Algo)

A comprehensive practice repository for mastering Data Structures and Algorithms through Jupyter Notebooks and problem-solving logs.

## Project Structure

- **Notebooks**:
  - `array.ipynb`: Fundamental array operations and problems.
  - `LinkedListProblems.ipynb`: Traversal, manipulation, and advanced problems.
  - `Sorting.ipynb`: Implementations of Quicksort, Mergesort, and others.
  - `Trees.ipynb`: Binary Tree traversals and Binary Search Tree logic.
- **Reference Sheets**:
  - `LinkedList.md`, `OOPS.md`, `Time_complexity.md`: Quick reference notes and theoretical concepts.
- **LeetCode Solutions**:
  - `leetcode/`: A specialized directory for competitive programming solutions.
- **Planning**:
  - `Map-plan.md`: Strategy for mastering algorithmic concepts.

## Key Topics
- Dynamic Programming
- Graph Theory
- Recursion & Backtracking
- Time & Space Complexity Analysis

## Getting Started
Open the notebooks using Jupyter Lab or a similar environment to review the implementations:
```bash
jupyter notebook array.ipynb
```
